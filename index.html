Clima

Acessar Site

Eu criei um sistema web incrível utilizando HTML, CSS e JavaScript. Com a integração da API OpenWeatherMap, o site exibe em tempo real o clima de diferentes países e cidades, proporcionando aos usuários informações precisas sobre o tempo. Além disso, adicionei outra API que exibe imagens de fundo relacionadas à busca do usuário, tornando a experiência ainda mais envolvente e personalizada. É gratificante ver como essas tecnologias se unem para criar uma plataforma prática e funcional para os usuários explorarem o clima do mundo de forma visualmente atraente. Link repositório
